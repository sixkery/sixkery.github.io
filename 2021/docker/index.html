docker 更换镜像源

Written by sixkery with ♥ on April 1, 2021 in docker , 160 words

在使用 docker 的过程中，拉取国外的镜像源十分慢，有时还会报错，可以更换为国内的镜像源。

可用的镜像源

https://registry.docker-cn.com Docker 官方中国区
http://hub-mirror.c.163.com 网易
https://docker.mirrors.ustc.edu.cn 中国科技大学

编辑 docker 的镜像源配置文件 vim /etc/docker/daemon.json（第一次操作可能不存在，新建就好）

{
    "registry-mirrors": ["https://registry.docker-cn.com","http://hub-mirror.c.163.com","https://docker.mirrors.ustc.edu.cn"]
 }

然后重启服务 service docker restart

查看是否生效 docker info|grep Mirrors -A 1

WARNING: No swap limit support
 Registry Mirrors:
  https://registry.docker-cn.com/

然后在进行拉取，不会报错。
